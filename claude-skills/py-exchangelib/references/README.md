# Exchangelib Skill References

This folder contains offline snapshots of the official documentation to support coding without internet.

Included:
- `webpage_exchangelib.md`: Main guide and usage documentation (installation, auth, folders, search, bulk ops, meetings, contacts, attachments, recurrence, OOF, rules, export/upload, sync, subscriptions, non-account services, troubleshooting, tests, contributing).
- `webpage_exchangelib API documentation.md`: API docs index with module/class/function signatures for exchangelib.

Original source: https://ecederstrand.github.io/exchangelib/
