# Prompt: Troubleshoot Common Issues

Goal: Identify and resolve frequent WinSW problems.

## Checklist
- Elevation/UAC and account rights
- Config path/name matching in bundled/global modes
- Log path permissions; rotation mode
- Shared directory mapping availability
- Encoding and redirection of stdout/stderr

## Output
- Root cause hypotheses with evidence
- Step-by-step fixes with references to [TROUBLESHOOTING.md](../TROUBLESHOOTING.md)
