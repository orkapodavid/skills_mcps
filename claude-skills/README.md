# Claude Skills

This directory contains skills for Claude Code and Claude Desktop, including utilities for MCP management, Python development, testing, and more.

## Available Skills

Browse the subdirectories to explore available skills:

- **mcp-management/**: Tools for managing and configuring MCP servers
- **project-planner-skill/**: Project planning and task breakdown utilities
- **skill-creator/**: Tools for creating new skills
- **skill-installer/**: Utilities for installing and managing skills
- **pytest/**: Python testing patterns and utilities
- **python-database-patterns/**: Database interaction patterns
- **python-packaging/**: Python package management utilities
- **async-python-patterns/**: Asynchronous programming patterns
- **data-pipeline-patterns/**: Data processing pipeline patterns
- **reflex-dev/**: Reflex framework development utilities
- **cli/**: Command-line interface development tools
- **microsoft-docs/**: Microsoft documentation access utilities
- **sqlserver-expert/**: SQL Server expertise and patterns

## Usage

Each skill directory contains:
- **SKILL.md** or **README.md**: Skill description and usage instructions
- **scripts/**: Implementation scripts
- **references/**: Supporting documentation
- **assets/**: Additional resources

## Interaction Guidelines

For guidelines on how to interact with Claude when using these skills, see [CLAUDE.md](CLAUDE.md).

## Adding New Skills

To add new skills, use the **skill-creator/** skill or follow the existing skill structure.
